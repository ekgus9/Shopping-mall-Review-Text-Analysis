# 감성사전을 활용한 아이패드와 갤럭시 탭 감성분석


 본 연구에서는 ‘아이패드(iPad)’와 ‘갤럭시 탭(Galaxy Tab)’의 사용자 리뷰에 대한 속성기반 감성분석(AbSA, Aspect-Based Sentiment Analysis)을 진행하고, 이를 통해 제품의 소비자 선호도를 비교 및 분석하고자 하였다.
